========================================
USING THE 5-text_indentation.py FUNCTION
========================================
This documentation serves as a guide to using the function.

Usage
=====
function that prints a text with 2 new lines after each of these
characters: ., ? and :
Args:
	text(str): string passed to function
text must be a string, otherwise raise a TypeError exception with
the message text must be a string

========
